## Instruções de como rodar a Aplicação

Para rodar a Aplicação em ambiente local é necessário ter instalado na máquina os seguintes recursos:

- NodeJs
- MongoDB
- Yarn ( opcional )

## Iniciando o Backend

### `cd backend`
### `npm install`
### `npm run dev`

## Iniciando o Frontend

### `cd frontend`
### `npm install`
### `npm run dev`
